### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/SamoZ256/lava_core.git
   ```
2. Enter the directory
   ```sh
   cd lava_core
   ```
3. Create a build directory
   ```sh
   mkdir build
   ```
4. Enter the build directory
   ```sh
   cd build
   ```
5. Initialize CMake
   ```sh
   cmake ..
   ```
5. ```sh
   make
   ```
